# Avoid Boxes 🎮

떨어지는 장애물을 피하며 생존하는 2D 아케이드 게임입니다.

## 🎯 게임 소개

Avoid Boxes는 플레이어가 화면 하단에서 좌우로 이동하며 떨어지는 장애물을 피하고 생존 시간을 늘리는 게임입니다. 다양한 스킬과 아이템을 활용하여 더 오래 생존하고 높은 점수를 획득하세요!

## ✨ 주요 기능

### 게임플레이
- **장애물 회피**: 다양한 타입의 장애물을 피하며 생존
- **웨이브 시스템**: 시간이 지날수록 난이도가 증가하는 웨이브 시스템
- **보스전**: 주기적으로 등장하는 보스를 처치하여 보너스 점수 획득
- **콤보 시스템**: 연속 회피 시 콤보가 쌓이며 점수 보너스 적용

### 스킬 시스템
- **숨기 (Space)**: 1초간 투명해져 장애물 통과 가능
- **무적 (Q)**: 5초간 무적 상태 (아이템 필요)
- **소형화 (W)**: 10초간 플레이어 크기 축소 (아이템 필요)
- **슬로우 모션 (E)**: 5초간 시간이 느려짐 (아이템 필요)
- **피버타임 (R)**: 콤보 100 이상 시 발동 가능, 10초간 장애물 속도 증가 (1.75배)

### 아이템 시스템
- **무적 아이템**: 무적 스킬 사용 가능 횟수 +1
- **소형화 아이템**: 소형화 스킬 사용 가능 횟수 +1
- **하트 아이템**: 생명 +1 회복
- **슬로우 모션 아이템**: 슬로우 모션 스킬 사용 가능 횟수 +1
- **공격력+1 아이템**: 발사체 공격력 영구 증가

### 보스 시스템
- **다양한 보스 타입**: 일반, 빠른, 큰, 분할 보스
- **웨이브별 HP 증가**: 웨이브가 올라갈수록 보스 체력 증가
- **발사체 공격**: A키로 발사체를 발사하여 보스 공격
- **보스 도달 시 게임 오버**: 보스가 플레이어 위치에 도달하면 게임 종료

### 커스터마이징
- **다크/라이트 테마**: 테마 전환 가능
- **게임 속도 조절**: 느림, 보통, 빠름 선택 가능
- **플레이어 색상 선택**: 8가지 색상 또는 자동 모드 선택

## 🎮 조작법

### 이동
- **← → (화살표 키)**: 좌우 이동

### 스킬
- **Space**: 숨기 스킬 발동
- **Q**: 무적 스킬 발동 (아이템 필요)
- **W**: 소형화 스킬 발동 (아이템 필요)
- **E**: 슬로우 모션 스킬 발동 (아이템 필요)
- **R**: 피버타임 발동 (콤보 100 이상)

### 공격
- **A**: 발사체 발사 (보스 공격용)

### 기타
- **ESC**: 일시정지/재개
- **P**: 게임 오버 후 재시작

## 📊 게임 통계

게임은 다음 통계를 저장합니다:
- 최고 점수
- 최고 생존 시간
- 최고 콤보
- 총 게임 플레이 횟수
- 총 플레이 시간

## 🏆 업적 시스템

다양한 업적을 달성하여 게임을 더욱 즐기세요:
- 점수 업적
- 생존 시간 업적
- 콤보 업적
- 보스 처치 업적

## 🛠️ 기술 스택

- **HTML5**: 게임 구조
- **CSS3**: 스타일링 및 애니메이션
- **JavaScript (Vanilla)**: 게임 로직 및 캔버스 렌더링
- **Canvas API**: 게임 그래픽 렌더링
- **LocalStorage**: 게임 통계 및 설정 저장

## 🚀 실행 방법

1. 저장소를 클론하거나 다운로드합니다:
```bash
git clone https://github.com/Socoms/Avoid-Boxes.git
```

2. `index.html` 파일을 웹 브라우저로 엽니다.

3. 게임을 시작합니다!
